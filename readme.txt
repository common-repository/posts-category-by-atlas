Плагин виджета для вывода постов в сайдбаре определенной рубрики

== Description ==

Плагин создания виджета для вывода постов (записей) определенной категории (Рубрики).
В виджете указывается заголовок, выбирается категория (можно выбрать категорию, у которой есть посты), записи которой
необходимо отображать в сайдбаре и указывается количество записей для вывода. Так же можно указать, чтобы при выводе постов в сайдбаре
указывалось время последней модификации

== Installation ==

1. Скопируйте папку плагина Posts category by Atlas в /wp-content/plugins/.
2. Активируйте плагин через меню Плагины.
3. Это все.
